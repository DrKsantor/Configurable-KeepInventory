# Configurable Inventory

Continuation of development mod Configurable Keep Inventory by Astemir.

This mod does not affect the vanilla /gamerule KeepInventory,
but adds a config that allows you to customize which slots should 
be kept after death.

CurseForge: https://www.curseforge.com/minecraft/mc-mods/configurable-keepinventory
